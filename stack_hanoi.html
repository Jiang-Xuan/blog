递归是一种自己调用自己的的算法，常见的递归就是斐波那契。

1 2 3 5 8 13 21 ...

下一个数等于前两个数之和。

我们来看看这个算法如何实现：

function fibonacci(n) {
	if(n == 1) {
		return 1
	} else if(n == 2) {
		return 2
	} else {
		return fibonacci(n - 1) + fibonacci(n - 2)
	}
}
